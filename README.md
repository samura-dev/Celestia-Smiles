# Celestia Smiles

Современный сайт стоматологической клиники на React + TypeScript с акцентом на премиальный визуальный стиль, 3D-композицию и продуманную UX-структуру.

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- React Three Fiber + Drei
- Zustand

## Локальный запуск

```bash
npm install
npm run dev
```

Приложение запускается на `http://localhost:4321`.

## Основные команды

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Структура проекта

- `src/pages` — страницы приложения
- `src/components` — UI и секции
- `src/data` — данные услуг, специалистов и отзывов
- `src/gl` — 3D-модели и сцены
- `src/styles` — глобальные стили и утилиты

## Продакшн-сборка

```bash
npm run build
```

Сборка формируется в папке `dist`.
