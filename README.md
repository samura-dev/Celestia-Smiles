# Celestia Smiles

Официальный сайт стоматологической клиники Celestia Smiles.

Проект представляет клинику, её направления лечения, команду специалистов и цифровой формат записи на приём. Интерфейс выполнен в премиальной визуальной стилистике с 3D-композицией, акцентом на читаемость, адаптивность и плавные пользовательские сценарии.

## Что есть на сайте

- Главная страница с презентацией клиники и ценностей
- Каталог направлений и детальные страницы услуг
- Раздел специалистов и карточки врачей
- Калькулятор ориентировочной стоимости лечения
- FAQ, юридические и информационные страницы
- Модальное окно записи на приём с выбором:
  - направления
  - врача по направлению
  - даты
  - доступного времени

## Технологический стек

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Three.js / React Three Fiber / Drei
- Zustand

## Архитектура проекта

- `src/pages` — страницы приложения
- `src/components` — UI-компоненты, layout, секции и модалки
- `src/data` — данные услуг, специалистов и отзывов
- `src/gl` — 3D-модели и сцены
- `src/styles` — глобальные стили и утилиты
